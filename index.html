About the Project

Concept

Hollyburn is a 2D top-down mystery game set at a summer camp that conceals a darker truth. Players investigate the camp's secrets through exploration, clue collection, and AI-powered NPC conversations.

Technology

Built in Godot 4 using the Sprout Lands asset pack. NPC dialogue is powered by the Eidolon AI SDK with per-character agent YAML configurations running on a local Docker server.

Research Focus

The project explores how AI-driven dialogue affects player moral reasoning — specifically whether dynamic, context-aware NPCs create more meaningful ethical dilemmas than scripted dialogue.

Repository

Source code available on GitHub at IainMacGraham/hollyburns. Godot 4.6 game engine with gdscripts aided by Claude.
